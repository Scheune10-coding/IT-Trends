## Windows-Rechner starten & einloggen

1. EC2 → Neue Instanz starten (Windows Server)
- .pem Datei zuweisen (Rest so lassen)

2. RDP-Port öffnen:
   - RDP (3389) → eigene IP

3. Administrator-Passwort entschlüsseln:
   - **Get Windows Password** → .pem hochladen

4. Per RDP verbinden:
   - Windows Remotedesktop → IP + Passwort eingeben
